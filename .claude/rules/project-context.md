\# Project Context



This project is a university web application for managing remote hackathons collaboratively.



The application allows users to:

\- create or join hackathons

\- create or join teams

\- manage team members

\- assign roles

\- create and follow tasks

\- monitor team progress



The project must remain:

\- realistic

\- technically coherent

\- maintainable

\- feasible for a student



Core entities:

\- Users

\- Hackathons

\- Teams

\- Team Members

\- Roles

\- Tasks

\- Join Requests

\- Comments (optional)

\- Messages (bonus)
